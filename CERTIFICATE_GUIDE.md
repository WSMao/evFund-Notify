# 區權證書功能說明

## 功能概述

這個功能可以為每位參與集資的住戶自動生成專屬的區權證書，並支援預覽和下載 PDF 格式的證書。

## 主要功能

### 1. 證書設計
- 採用類似獎狀的精緻設計
- 包含金色邊框和裝飾性角落
- 顯示以下資訊：
  - 住戶（區權人）姓名
  - 戶號
  - 車位號（樓層 + 編號）
  - 出資金額
  - 頒發日期
  - 社區印信

### 2. 證書管理頁面
- 路徑：`/certificates`
- 顯示所有集資參與者的清單
- 每位參與者旁邊都有：
  - 👁️ **預覽按鈕**：在彈出視窗中預覽證書
  - 📥 **下載 PDF 按鈕**：將證書轉換為 PDF 並下載

### 3. 技術實現
- 使用 **CSS** 設計精美的證書樣式
- 使用 **html2canvas** 將 HTML 證書轉換為圖片
- 使用 **jspdf** 將圖片轉換為 PDF 文件
- 支援高解析度輸出（2x scale）

## 使用方式

### 訪問證書管理頁面
1. 點擊導航欄的「查看方案」下拉選單
2. 選擇「區權證書」選項
3. 進入證書管理頁面

### 預覽證書
1. 在列表中找到要查看的住戶
2. 點擊「👁️ 預覽」按鈕
3. 在彈出的模態視窗中查看證書
4. 可以直接在預覽視窗中下載該證書

### 下載證書 PDF
1. 點擊任一住戶旁的「📥 下載 PDF」按鈕
2. 系統會自動生成 PDF（顯示「⏳ 生成中...」）
3. 完成後會自動下載，檔名格式：`區權證書_住戶名_戶號.pdf`

## 技術細節

### 依賴套件
```json
{
  "html2canvas": "^1.4.1",
  "jspdf": "^2.5.2",
  "@types/html2canvas": "^1.0.0"
}
```

### PDF 生成流程
1. 創建臨時的 DOM 元素包含證書 HTML
2. 使用 html2canvas 將 DOM 轉換為 Canvas
3. 將 Canvas 轉換為圖片（PNG）
4. 使用 jsPDF 創建 PDF 並插入圖片
5. 自動下載 PDF 文件
6. 清理臨時 DOM 元素

### 證書尺寸
- 寬度：800px
- 高度：600px
- 方向：橫向（landscape）
- 輸出：2x 解析度以確保清晰度

## 樣式特色

- **邊框**：深棕色主邊框 + 金色裝飾邊框
- **背景**：米色漸層背景
- **字體**：使用襯線字體（Noto Serif TC, PMingLiU）
- **裝飾**：四角裝飾框 + 陰影效果
- **印章**：圓形紅色印章效果（旋轉 -15度）

## 注意事項

1. PDF 生成需要幾秒鐘時間，請耐心等待
2. 證書內容直接從集資資料自動填入
3. 支援響應式設計，在移動裝置上也能正常使用
4. 證書樣式可以根據需要進一步客製化
